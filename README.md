# Survivor

「サバイバー」

（概要・テーマ）
この作品は、初めてDirectX9を使い開発したゲームです。
キャラクターの絵は全て自分で作った画像です。
この画像たちは個人的に気に入っています。

(開発時期・期間)
HAL東京2年生時の2017年8月～2017年9月(2ヶ月)

（開発ツール）
・開発環境：Visual Studio2015
・言語：C
・API：DirectX9
・2Dデータ制作：ミニドット絵メーカー2
・2D データの一部はフリー素材を
使用しています。  

（チームメンバー）
プログラマー1人（自分）

（ゲームシステム）
一定時間後に来る脱出用の茶色い車がやって来るまで敵の攻撃を避けるゲームです。
・敵の弾や爆弾に当たるとゲームオーバーになります。
・茶色い車に乗る（当たる）とゲームクリアーになります。

(操作方法)
・AとDキーでプレイヤーキャラクターを左右移動させることができます。
・Wキーでプレイヤーキャラクターをジャンプさせることができます。

(起動方法)
「サバイバー(ビルド)」というフォルダーを開いてください。
その中にある、「Survivor.exe」というアプリケーションをダブルクリックしてください。
